  1. По тегу
  2. По классу - самый нужный селектор
  3. По идентификатору ID
  4. * - выбор всех элементов.

Cелекторы (продолжение):

  1. Можно выбрать несколько селекторов через запятую.
  2. Выбор элемента ВНУТРИ другого элемента.
  3. p.class1 - выбирает элементы p с классом class1
  4. div + p - Выбирает все элементы p, которые расположены сразу после div.
