О Шрифтах

1

Как изменить шрифт?

Шрифты меняют с помощью CSS используя свойство font-family.

2

Альтернативный текст

Альтернативный текст указывается в свойстве font-family после запятой

3

Семейство шрифтов

Каждое семейства состоит из большого количества шрифтов, которые обладают общими характеристиками. Существует пять семейств: Serif, Sans-Sefir, Monospace, Cursive, Fantasy.

4

Google fonts

Используйте шрифты из сервиса Google fonts, чтобы не переживать найдутся ли нужные шрифты на компьютере других пользователей.
